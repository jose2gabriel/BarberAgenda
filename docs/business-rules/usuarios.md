# Regras de Negócio - Usuários

## Cadastro

* O usuário deve informar nome, e-mail, telefone e senha.
* O e-mail deve ser único no sistema.
* Não é permitido o cadastro de e-mails duplicados.
* As senhas devem ser armazenadas de forma criptografada.

## Autenticação

* O login deve ser realizado utilizando e-mail e senha.
* O sistema deve utilizar autenticação baseada em token.
* O token deve possuir prazo máximo de validade de 24 horas.

## Perfil

* O usuário pode atualizar seus dados cadastrais.
* O usuário pode consultar seus agendamentos.
* O usuário pode encerrar sua sessão a qualquer momento.

## Privacidade

* O usuário pode solicitar a exclusão de seus dados em conformidade com a LGPD.
