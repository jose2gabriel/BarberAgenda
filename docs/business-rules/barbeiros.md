# Regras de Negócio - Barbeiros

## Cadastro

* O sistema deve permitir o cadastro de barbeiros.
* Cada barbeiro deve possuir informações profissionais cadastradas.

## Agenda

* Cada barbeiro possui uma agenda individual.
* As agendas devem permanecer independentes entre si.
* O sistema deve permitir a visualização dos horários disponíveis por profissional.

## Disponibilidade

* O barbeiro pode possuir períodos de indisponibilidade.
* Períodos de indisponibilidade devem bloquear novos agendamentos.
* O sistema deve impedir conflitos de horários.

## Atendimento

* Todo agendamento deve estar associado a um barbeiro específico.
* O sistema deve informar ao cliente qual profissional realizará o atendimento.
